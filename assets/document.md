# Warunki wygaszeń systematyczych w grupie Fm-3m

Związek `FeO` należy do grupy przestrzennej nr 225 (Fm-3m).
Ta grupa prezentuje następujące warunki **odbicia**:

```{image} reflection_conditions.png
```

```
HKL :    H+K,H+L,K+L=2n+1
0KL :    K=2n+1, L=2n+1
H0L :    H=2n+1, L=2n+1
HK0 :    H=2n+1, K=2n+1
HHL :    H+L=2n+1
HKH :    H+K=2n+1
HKK :    H+K=2n+1
H00 :    H=2n+1
0K0 :    K=2n+1
00L :    L=2n+1
0KK :    K=2n+1
H0H :    H=2n+1
HH0 :    H=2n+1
```
